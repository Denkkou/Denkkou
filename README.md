I'm Joe and I'm a games computing graduate turned backend software engineer. I love to work in Python and Go due to their intuitive syntax and how they allow for fast prototyping. I have also an educational history utilising C and C++, and have developed an interest in the [Zig programming language](https://ziglang.org/) for its strict rules that make bugs take effort to produce.

Currently redesigning my Grandia save editor software, GrandiEdit, and moving out of Python and into Golang. Go is my current favourite language and it also serves the needs of the program better, being lower level.

I'm currently exploring SQL and databases in further detail, having a few years between me and my last encounter with them.

I'm looking to be more involved in open-source contributions and support that space in any way I can as I use many open-source alternatives for both work and play. I believe in its mission and try to make all of my projects open for use too!
