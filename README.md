I'm Joe and I'm a BSc. Games Computing graduate (1st) turned backend software engineer.

I have a keen interest in the low-level (or more generally, anything 'behind the scenes'), being driven primarily by curiousity and discovery. I have built web utilities such as scrapers, static site generators and REPL CLI tools for interacting with APIs in Python and Golang, as well as small game engines, emulators, and save-game binary editors in C, C++ and Zig.

Current learning endevours: Haskell & FP, Cryptography and HTTP servers in Golang, PSX emulation in C++.

Why these languages?
- Golang combines everything I love about low-level programming with the modern legibility of languages like Python. I find it very quick to write and parse, and for programs where I don't need to optimise performance, a bit of garbage collection never hurts :]
- Zig makes it difficult to make common C++ mistakes; I love how it enforces a sort of 'correct way to go about it' when writing programs. That said, C++ doesn't abstract much away, keeping me close to the metal when it matters.

I'm looking to be more involved in open-source contributions!
